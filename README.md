# Carnival of Cultures

A premium, component-based website for showcasing cultural clubs, events, and galleries.

## Features
- **Dynamic Landing Page**: Showcases all clubs with a premium glassmorphic UI.
- **Club Details**: Dedicated pages for Music, Dance, Drama, Art, Literature, and Photography clubs.
- **Dynamic Routing**: Uses React Router for seamless navigation.
- **Responsive Design**: Fully responsive layout with mobile menu.
- **Interactive UI**: Hover effects, smooth transitions, and glassmorphism.

## Tech Stack
- React (Vite)
- Vanilla CSS (CSS Modules / Styled via JS objects & index.css)
- Lucide React (Icons)
- React Router DOM

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
